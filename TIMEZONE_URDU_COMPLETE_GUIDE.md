
## 🎓 Developers کے لیے

### کیسے استعمال کریں:

```javascript
// Import کریں
import { formatToLocalTime } from '@/utils/timezone';

// استعمال کریں
const localTime = formatToLocalTime(ticket.created_at);
```

### تمام Functions:

1. **`formatToLocalTime(date)`** - مکمل تاریخ اور وقت
2. **`formatDateOnly(date)`** - صرف تاریخ
3. **`formatTimeOnly(date)`** - صرف وقت
4. **`formatActivityLogDate(date)`** - Activity logs کے لیے
5. **`getUserTimezone()`** - User کا timezone نام
6. **`getTimezoneOffset()`** - UTC سے فرق
